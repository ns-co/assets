6.0 Appendix

Anything that did not fit in the body of the manual can be found here: addenda, updates, credits, special notes and special thanks to our sponsors, supporters, partners and fans!

6.1 Reference Documentation

These ASCII documents contain the latest reference, support and configuration information about NS:CO and are updated automatically when you apply any patches or updates to NS:CO, they are viewable directly in the Manual.


Quick Start Guide: a brief introduction to the basics you will need to play and enjoy NS:CO, a must read for first time players.
 

NSSL Scripting Guide: a guide that covers all the basics of the NSSL scripting language unique to NS:CO.
 

Skill Instruction Sheet: a detailed reference of every NS:CO TASS™ ability or skill and how each additional additional experience point effects game-play.
 

Commands: documents the any late additions or changes to NS:CO commands and binds, how they are used and function.

CVARS: a comprehensive reference of the major NS:CO console variables and their functions.
 

Level Designers Reference Sheets: a directory of reference material for Level Designers that want to map for Navy SEAL/s: Covert Operations.
